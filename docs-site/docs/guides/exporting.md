# Exporting

```bash
lore export <format> [--out <dir>] [--json]
```

## Formats

| Format | Description |
|---|---|
| `bundle` | Single concatenated .md file |
| `slides` | Marp-compatible .md |
| `pdf` | Puppeteer-rendered PDF |
| `docx` | Structured DOCX |
| `web` | Astro Starlight or VitePress project |
| `canvas` | JSON Canvas 1.0 |
| `graphml` | GraphML XML for Gephi/yEd |
